# Software Project Management System (SPMS)

## Technologies
- Python
- Tkinter
- MySQL
- GitHub
- Unit Testing

## Features
- User Authentication
- Project CRUD
- Task CRUD
- Clean Architecture
- Unit Tests

## How to Run
1. Import `schema.sql` into MySQL
2. Update DB credentials in `config.py`
3. Install requirements
4. Run `main.py`
